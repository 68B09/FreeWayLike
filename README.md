# FreeWayライクとは
そのむかし、FKSさんが作成されたPC-9801シリーズで動作する「FreeWay」という環境アプリがあり、Yoshihiro Sugahara氏がそれのスクリーンセーバー版を作成されました。  
「FreeWayライク」はそれらをJavaScriptで模倣したものです。  
元祖FreeWayおよびスクリーンセーバー版のソースコードが見つからなかったため「FreeWayライク」のコードはイチから作成しました。  
そのため動作は元祖及びスクリーンセーバー版とは異なりますが、なんとなくそれっぽい動作にはなっているかな？と思います。  
なにかの参考になれば幸いです。  

本リポジトリの著作権については、FKS氏とYoshihiro Sugahara氏に帰属します。
